## Creating Interactive Components 

I came across <a this news story href="https://www.tdtnews.com/news/central_texas_news/article_dc55d5ce-8a66-5cba-8dcd-2baf7cf932c9.html"> from a local publication in Texas that listed out all the local polling places for voters in the primaries. I took one look at the website and said, wow that's unhelpful.

I worked with CoPilot (read: CoPilot in planning mode did a lot of heavy lifting) to take the addresses listed in the Temple Daily Telegram's reporting to create a tool that lets users put in their address or zip code and the component will tell them where their nearest polling location.
